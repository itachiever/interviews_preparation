# Offshore AgentOps & DevSecOps Engineer

**Company:** Intellectyx Data Science India Private Limited
**Department:** Engineering
**Experience:** 5-7 Years
**Employment Type:** Full Time
**Location:** India

We are seeking an Offshore AgentOps & DevSecOps Engineer to design, secure, deploy, and operate AI agents and cloud-native applications at scale across Intellectyx's AI portfolio.

This role blends Agent Operations (LLM / AI Agents) with DevSecOps best practices, with AWS as the primary cloud.

## Key Responsibilities — AgentOps (AI / LLM Operations)

* Deploy, monitor, and maintain AI agents, LLM pipelines, and agentic workflows built on LangChain, LangGraph, AgnoAI, MCP in production.
* Implement versioning, observability, logging, tracing, and performance monitoring using Langfuse, OpenTelemetry, Prometheus, Grafana, AWS CloudWatch and X-Ray.
* Manage prompt lifecycle, model evaluation, fallback mechanisms, guardrails, and agent governance.
* Ensure scalability, reliability, and cost optimization (token usage, model routing, caching).
* Integrate agents with enterprise systems and data sources — primarily on AWS (S3, Redshift, Glue, Lambda, EventBridge, API Gateway, RDS/PostgreSQL, Bedrock), Snowflake / Databricks on AWS, ERP and CRM — securely.

## Key Responsibilities — DevSecOps

* Design and implement secure CI/CD pipelines with embedded security checks.
* Automate infrastructure provisioning using IaC — Terraform and AWS CloudFormation / CDK primary.
* Implement SAST, DAST, SCA, secrets management, and container security.
* Ensure compliance with organizational and regulatory standards (SOC 2, ISO 27001, GDPR).
* Perform vulnerability assessments, remediation, and security hardening across AI and data workloads.

## Key Responsibilities — Cloud & Platform Engineering

* Manage workloads primarily on AWS (EKS, ECS, Lambda, EC2, Fargate, API Gateway, CloudWatch, CloudTrail, IAM, KMS, Secrets Manager, WAF, GuardDuty, Bedrock, SageMaker); Azure / GCP valuable for other clients.
* Deploy and manage Docker, Kubernetes, and service meshes.
* Implement monitoring using AWS-native tooling (CloudWatch, X-Ray, OpenSearch) alongside Prometheus, Grafana, ELK, Datadog, Langfuse.
* Optimize system reliability through HA, DR, and resiliency — including inference fallback and multi-model routing.

## Key Responsibilities — Collaboration & Operations

* Work closely with AI engineers, product teams, architects, and security teams.
* Provide operational support, incident response, and root-cause analysis at both application and agent levels.
* Lead offshore execution and ensure high-quality, on-time deliverables.
* Document processes, runbooks, and architectural designs.

## Required Skills & Qualifications

* Strong hands-on experience in DevSecOps practices and tools.
* Experience with AI / LLM platforms — AWS Bedrock (Claude, Llama, Titan), Anthropic, OpenAI, Hugging Face, LangChain, LangGraph, AgnoAI, MCP, LlamaIndex. Azure OpenAI a plus.
* Proficiency in Python, Bash, or Golang.
* Expertise in CI/CD — GitHub Actions and AWS CodePipeline / CodeBuild primary; GitLab CI, Jenkins, Azure DevOps acceptable.
* Strong knowledge of cloud security, IAM, and zero-trust principles.
* Experience with containers & orchestration (Docker, Kubernetes).
* Knowledge of API security, OAuth, secrets management — AWS KMS, Secrets Manager, Parameter Store; HashiCorp Vault also valuable.
* Familiarity with LLM-focused observability (Langfuse, Arize, WhyLabs).
