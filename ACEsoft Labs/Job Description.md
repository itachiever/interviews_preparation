## Job Description

- Design and implement security controls for containerized applications and cloud-native environments on Azure across development and production stages.
- Conduct threat modeling, secure code review, and vulnerability assessments for web applications, APIs, and microservices running on Docker and Kubernetes.
- Implement identity and access management, workload identity federation, encryption, and key management practices in Azure cloud environments.
- Design, set up, and maintain CI/CD pipelines with integrated security tooling including SAST, DAST, SCA, and container image scanning at each pipeline stage.
- Monitor cloud configurations and Kubernetes posture using CSPM and CWPP tools to continuously detect and remediate misconfigurations and runtime risks in Azure environments.
- Enforce compliance with frameworks such as ISO 27001, NIST, and CIS Benchmarks for Kubernetes cluster hardening, container workloads, and Azure cloud deployments.
- Investigate and respond to security incidents involving containerized applications, APIs, Kubernetes workloads, or Azure cloud resources.
- Implement Zero Trust and least privilege principles across application access, Kubernetes RBAC, pod security standards, and Azure IAM policies.
- Provide guidance on secure application architecture, container security hardening, and data protection strategies for cloud-native workloads on Azure.
- Write Python and Bash scripts to automate security workflows, including vulnerability triage, remediation tracking, alert enrichment, and security reporting.
- Review and enforce security best practices in Infrastructure as Code (Terraform/Bicep) templates, including policy-as-code checks in deployment pipelines.
- Build and maintain custom security tooling and integrations to extend detection, response, and reporting capabilities across the product security ecosystem.
- Manage software supply chain security by enforcing dependency scanning, SBOM generation, container image provenance, and third-party component risk assessments across product pipelines.
- Maintain dashboards, runbooks, and security metrics to report product and cloud security maturity, pipeline posture, and vulnerability trends to engineering leadership.
