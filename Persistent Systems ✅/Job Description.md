# Job Description: DevSecOps / Security Engineer

**Location:** Bangalore (Hybrid)  
**Experience:** 4 - 6 Years

## Role Overview

We are looking for a skilled DevSecOps / Security Engineer with strong expertise in integrating security practices into CI/CD pipelines and DevOps processes. The role focuses on security engineering, automation, and vulnerability management, ensuring that applications and infrastructure are secure across the SDLC.

## Key Responsibilities

- Integrate and embed security tools and controls into CI/CD pipelines using platforms such as Jenkins, GitHub Actions, and GitLab.
- Implement and manage security testing mechanisms including:
  - SAST (Static Application Security Testing)
  - DAST (Dynamic Application Security Testing)
  - Secret scanning and dependency/security checks
- Enable and manage SBOM (Software Bill of Materials) generation, tracking, and governance to ensure supply chain security.
- Drive shift-left security practices by integrating security early in the development lifecycle.
- Automate security checks across build and release pipelines to enforce compliance and reduce vulnerabilities.
- Manage and enhance vulnerability assessment and remediation processes across applications and infrastructure.
- Collaborate with DevOps, development, and security teams to ensure secure software delivery practices.
- Develop and maintain automation scripts using Python and PowerShell for security tooling and operational workflows.
- Configure and maintain security dashboards, reports, and metrics to track security posture and risks.
- Integrate security tools with SCM systems to enable automated issue tracking and remediation workflows.
- Support security operations activities, including monitoring, incident response inputs, and governance alignment.

## Required Skills & Qualifications

- Strong experience in DevSecOps and Security Engineering.
- Hands-on experience integrating security tools into CI/CD pipelines:
  - Jenkins
  - GitHub Actions
  - GitLab CI
- Expertise in application security tools:
  - SAST
  - DAST
  - Secret scanning
  - Software Composition Analysis (SCA) / SBOM
- Experience in vulnerability assessment and remediation lifecycle.
- Strong scripting expertise in:
  - Python
  - PowerShell
- Good understanding of DevOps practices and SDLC.
- Knowledge of security operations and compliance practices.
- Familiarity with version control systems (Git).

## Preferred Skills (Good to Have)

- Experience with security tools like Mend.io, SonarQube, or similar platforms.
- Exposure to cloud environments (AWS / Azure).
- Knowledge of container security (Docker, Kubernetes security practices).
- Understanding of Zero Trust and secure architecture principles.
- Experience working with SOC/NOC teams or security operations workflows.
