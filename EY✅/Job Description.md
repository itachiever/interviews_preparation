# Role – DevSecOps Engineer

**Location:** Bangalore (Hybrid Mode)  
**Experience:** 4+ Years

# Job Description

We are seeking a DevSecOps Engineer to design, implement, and operate secure, automated, and scalable CI/CD platforms on Microsoft Azure. The role embeds security controls across the entire cloud and application lifecycle, while enabling development teams to deliver software rapidly and safely.

The ideal candidate has strong hands-on experience with:
- Azure cloud services
- CI/CD pipelines
- Infrastructure automation
- Security engineering

The candidate should have a mindset that balances:
- Speed
- Resilience
- Risk management

# Key Responsibilities

## 1. DevSecOps Platform & CI/CD Engineering

Design, build, and maintain CI/CD pipelines using:
- Azure DevOps
- GitHub Actions

Integrate:
- Security scanning
- Policy enforcement
- Compliance checks into pipelines

Implement automated:
- Build processes
- Test processes
- Deployment processes
- Rollback processes

Manage:
- Artifact versioning
- Release governance

Enable:
- Shift-left security practices across development teams

## 2. Azure Cloud Infrastructure & Automation

Design and implement secure cloud infrastructure using:
- Azure Resource Manager (ARM)
- Bicep
- Terraform

Manage and secure Azure resources including:
- Virtual Networks (VNETs)
- Subnets
- NSGs
- Azure Load Balancer
- Application Gateway
- Private Endpoints
- Azure Storage
- Key Vault
- Azure SQL / Cosmos DB

Support:
- Multi-environment deployments (Dev, UAT, Prod)

## 3. Security Engineering & Controls

Embed security into cloud and application lifecycle:
- Infrastructure-as-Code (IaC) security scanning
- Static Application Security Testing (SAST)
- Dependency scanning
- Container scanning

Implement and manage:
- Azure Key Vault (secrets, certificates, keys)
- Managed Identities
- RBAC

Use Azure-native security services:
- Microsoft Defender for Cloud
- Microsoft Sentinel (SIEM/SOAR)
- Azure Policy
- Azure Blueprints

## 4. Container & Platform Security

Secure containerized workloads using:
- Azure Kubernetes Service (AKS)
- Azure Container Registry (ACR)

Implement:
- Image scanning
- Admission controls
- Pod security
- Network policies
- Secrets management

Support:
- Runtime security
- Vulnerability remediation

## 5. Monitoring, Reliability & Operations

Implement observability and monitoring using:
- Azure Monitor
- Log Analytics
- Application Insights

Define and track:
- SLIs
- SLOs
- Error budgets

Support:
- Incident response
- Root cause analysis
- Post-incident reviews

## 6. Governance, Risk & Compliance

Enforce cloud governance using:
- Azure Policy
- Management Groups
- Secure landing zones

Support compliance requirements:
- Internal security standards
- Regulatory and audit requirements
- Financial services compliance
- Data protection standards

Collaborate with:
- Security teams
- Architecture teams
- Risk and Compliance stakeholders

# Qualifications

- Minimum 5 years of IT industry experience
- Strong exposure to:
  - Azure Cloud
  - DevSecOps
  - Infrastructure Engineering

Preferred Certification:
- Microsoft Certified: AZ-400 DevOps Engineer Expert

Preferred Education:
- Bachelor’s degree in Computer Science, Engineering, or related field
