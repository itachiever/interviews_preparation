# Role : Jenkins Engineer 

-

Job description
Role & responsibilities

Key Responsibilities

1. CI/CD Pipeline Management

Manage existing Jenkins pipelines for onpremise server deployments.
Monitor, troubleshoot, and optimize pipeline performance and reliability.
Maintain pipeline-as-code repositories (Jenkinsfilebased).
2. Automation Framework Enhancements

Enhance and maintain automated code check stages (linting, security scans, unit test triggers, etc.).
Integrate tools such as SonarQube, Checkmarx, or other static code analysis utilities based on project needs.
Work closely with developers and DevOps leads to improve automation logic.
3. New Pipeline Development

Design and build CI/CD pipelines based on new data or deployment requirements.
Implement declarative or scripted Jenkins pipelines as needed.
Integrate with GIT, Bitbucket, or other SCM tools.
4. Deployment & Environment Support

Manage build and deployment workflows for onpremise environments.
Coordinate with Infra teams for agent management, node configurations, and access provisioning.
Maintain security and compliance standards in pipeline execution.
Required Skills

Strong knowledge of Jenkins, including declarative and scripted pipelines.
Familiarity with Linux/Windows environments used in onprem deployments.
Handson experience with version control systems (Git).
Exposure to DevOps tools such as SonarQube, Ansible, Terraform, Docker, Kubernetes (optional but beneficial).
Knowledge of shell scripting (Bash/PowerShell).
Ability to troubleshoot pipeline failures and optimize CI/CD workflow.
