"Our architecture follows a hybrid model where DevSecOps tools such as GitLab, Jenkins, Nexus, SonarQube, and Fortify are hosted on-premises, while applications run on Azure AKS. Developers push code to GitLab, which triggers Jenkins pipelines. Jenkins performs build, code quality checks, security scans, Docker image creation, and artifact publishing. For deployments, Jenkins communicates through an NGINX server hosted in the DMZ and then over a secure VPN tunnel to the private AKS cluster. Application traffic flows from users through Azure Load Balancer, Kong Ingress, Kubernetes Services, and finally to application pods. Monitoring is handled through Prometheus, Grafana, ELK, and Azure Monitor, while secrets are managed through Azure Key Vault. This architecture provides secure deployments, network isolation, centralized monitoring, and compliance for enterprise applications."
